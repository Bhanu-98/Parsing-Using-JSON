
Step 1: Create a Database with name 'mavoix';
Step 2: Run MYSQL Script Provided to create tables.
Step 3: Insert data into two tables, appointment and patient. Leave the 'apn' empty.
Step 4: Open project as Maven and build it. Make sure File Location is properly given in the To classes.
Step 5: Run DataBaseToJSON class in appointment package to get details of appointment for a patient with mobile number.            
	Run JSONToDataBase class in appointment package to insert appointment details to database for a given mobile number.           
        Run DataBaseToJSON class in patient package to get details of patient with mobile number.
